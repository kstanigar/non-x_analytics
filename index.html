Step 1 — Register GA4 Custom Dimensions
Go to GA4 Admin → Property → Custom Definitions → Custom Dimensions → Create. Register all of these as event-scoped dimensions:
Parameter NameScopeDescriptionUsed In Events
ab_music_groupEventMusic A/B cohort — 'music_on' or 'music_off'All 16 events
music_variantEventActual music state at event timeAll in-game events
platformEvent'desktop' or 'mobile'All in-game events
level_numberEventLevel reached (1–12)wave_reached
phaseEvent'green' / 'red' / 'purple' difficulty phasewave_reached, player_death
level_reachedEventHighest level before deathplayer_death

Also register these as Custom Metrics (numeric): session_duration_seconds, score
Step 2 — Calculated Fields (Looker Studio Syntax)
Create these in your Looker Studio data source under Add a field → Calculated field:
PLAYS
SUM(CASE WHEN event_name = "game_start" THEN event_count ELSE 0 END)
WINS
SUM(CASE WHEN event_name = "player_won" THEN event_count ELSE 0 END)
DEATHS
SUM(CASE WHEN event_name = "player_death" THEN event_count ELSE 0 END)
WIN RATE
SAFE_DIVIDE( Wins, Plays )
BOSS 1 DEFEATS
SUM(CASE WHEN event_name = "boss_defeated" AND boss_id = 1 THEN event_count ELSE 0 END)
BOSS 1 CONVERSION
SAFE_DIVIDE( Boss1Defeats, Plays )
BOSS 2→3 CONVERSION
SAFE_DIVIDE( Boss2Defeats, Boss1Defeats )
LEADERBOARD RATE
SAFE_DIVIDE( LeaderboardSubmits, Wins )
